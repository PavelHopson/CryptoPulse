# Быстрый старт (RU)

## Требования
- Node.js 20+
- npm
- аккаунт Supabase
- аккаунт Stripe (для платежей)

## Запуск
1. Установить зависимости:
   ```bash
   npm install
   ```
2. Создать `.env`:
   ```bash
   cp .env.example .env
   ```
3. Заполнить переменные окружения.
4. Запустить:
   ```bash
   npm run dev
   ```

## Проверки
```bash
npm run lint
npm run test
npm run build
```
